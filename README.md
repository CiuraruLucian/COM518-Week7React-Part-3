# React Context, Maps, and API Search Exercises

This repository contains a set of exercises completed in class to practice key React concepts including:

- Component hierarchy and prop passing
- React Context API
- State lifting
- useEffect and asynchronous API requests
- Integrating interactive maps with Leaflet
- Displaying search results on both a map and a list

The exercises gradually evolve a React application from a simple prop-based structure into a more scalable architecture using Context and reusable components.

---

## Learning Objectives

The goal of these exercises is to understand how to:

1. Pass state through multiple component layers using props
2. Replace prop drilling with React Context
3. Separate UI functionality into reusable components
4. Fetch external data using `useEffect`
5. Integrate mapping functionality using Leaflet
6. Synchronize application state between multiple components
7. Visualize data both as UI elements and map markers
